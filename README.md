📝 Todo App

A simple and responsive Todo App built to help users manage daily tasks efficiently. Users can add, edit, complete, and delete tasks with a clean and user-friendly interface.

🚀 Features

- ➕ Add new tasks
- 🗑️ Delete tasks
- 📋 View all tasks
- 📱 Responsive design
- ⚡ Fast and easy to use

🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)

«If your project uses React, Tailwind CSS, or any other technology, update this section accordingly.»

📂 Project Structure

todo-app/
├── index.html
├── style.css
├── script.js
└── README.md

📦 Installation

1. Clone the repository

git clone https://github.com/your-username/todo-app.git

2. Navigate to the project folder

cd todo-app

3. Open "index.html" in your browser.

🎯 How to Use

1. Enter a task in the input field.
2. Click the Add button.
3. Delete tasks you no longer need.
